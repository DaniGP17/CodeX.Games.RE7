# CodeX (030) — Resident Evil 7 module

Research project that adds **Resident Evil 7 Biohazard** support to [CodeX](https://www.patreon.com/dexyfex) by dexyfex. Lets you browse the game's PAK archives, preview RE Engine assets (meshes, textures, scenes), and walk through whole chapters.

## Features

- **PAK browser** — read RE7's KPKA `.pak` archives.
- **Texture viewer** — `.tex` decoded to a per-mip / per-slice viewer.
- **Mesh viewer** — `.mesh` preview of 3d models including materials.
- **Scene viewer** — `.scn` parsed via RE Engine RSZ system (folders, gameobjects, components, transforms).
